About Our Project

Our project is a platform developed to assist users in an effective way in managing and planning their custom PC builds. Featuring a friendly user interface based on standard Material UI, the app allows users to input components details, manage different PC build configurations, check its compatibility and save data through a database.This project aims to provide a centralized tools which enables users to easily organize, compare and optimize PC configurations, ensure that both regular users and tech experts has a comfortable experience when using this app.
